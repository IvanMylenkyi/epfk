



Щоб запустити основну програму, виконайте в терміналі:
python electricity_calculator.py

У проєкті реалізовано як `doctest`, так і `unittest`.

1. **Запуск Unit-тестів:**

python -m unittest tests.py


2. **Запуск Doctests:**
Doctests запускаються автоматично при виконанні основного файлу. Щоб побачити детальний звіт про проходження `doctests`, запустіть:

python -m doctest -v calc.py

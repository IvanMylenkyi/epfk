
## Вимоги
- Python 3.8+
- Акаунт на [sentry.io](https://sentry.io/)

## Встановлення та запуск

1. **Встановіть залежності**:
   ```bash
   pip install sentry-sdk
2. Налаштуйте Sentry:
Отримайте свій DSN у налаштуваннях проекту Sentry та встановіть його як змінну оточення:

Windows (CMD): set SENTRY_DSN=ваш_dsn_посилання

Linux/macOS: export SENTRY_DSN='ваш_dsn_посилання'

3. Запустіть програму:

python main.py
